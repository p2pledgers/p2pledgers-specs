## Build

Use `bin/make.sh` to build the pdf using `Pandoc` without argument clutter:

```
git clone git@github.com:p2pledgers/p2pledgers-specs.git
cd p2pledgers-specs
./bin/make.sh src/*.md -o build/p2pledgers-specs.pdf
```
